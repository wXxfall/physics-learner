# arXiv Beautifier

## 项目简介

arXiv Beautifier 是一个浏览器扩展，用于美化 arXiv 网站的界面，提供更美观、现代的阅读体验。该扩展通过注入自定义 CSS 和 JavaScript，在不修改原始网站功能的情况下，为 arXiv 网站提供一个视觉上更赏心悦目的"外壳"。

## 技术方案

### 实现原理
1. **浏览器扩展**：使用 Chrome 扩展（Manifest V3）技术
2. **内容脚本**：通过 Content Script 注入 CSS 和 JavaScript
3. **实时同步**：使用 MutationObserver 监听 DOM 变化，保持内容实时更新
4. **美化容器**：创建美化容器，将原始内容复制到容器中并应用美化样式

### 项目结构

```
arxiv/
├── manifest.json       # 扩展配置文件
├── content.js          # 核心功能实现
├── styles.css          # 美化样式
├── icons/              # 图标文件夹
│   ├── icon16.png      # 16x16 图标
│   ├── icon48.png      # 48x48 图标
│   └── icon128.png     # 128x128 图标
└── README.md           # 项目说明
```

## 安装方法

### Chrome/Edge 浏览器

1. 打开浏览器扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

2. 启用"开发者模式"

3. 点击"加载已解压的扩展程序"

4. 选择 `arxiv` 文件夹

5. 扩展安装成功后，访问 arXiv 网站即可看到美化效果

### Firefox 浏览器

1. 打开 Firefox 扩展管理页面：`about:debugging`

2. 点击"此 Firefox"

3. 点击"临时载入附加组件"

4. 选择 `manifest.json` 文件

5. 扩展临时安装成功，访问 arXiv 网站即可看到美化效果

## 实现细节

### 核心功能

1. **初始化美化**：
   - 创建美化容器
   - 复制原始内容到容器中
   - 应用美化样式

2. **实时同步**：
   - 使用 MutationObserver 监听 DOM 变化
   - 当内容变化时，自动重新应用美化

3. **美化处理**：
   - 美化论文列表
   - 美化论文详情页
   - 美化搜索结果

4. **错误处理**：
   - 捕获并处理异常
   - 确保扩展稳定运行

### 样式特性

- **现代设计**：使用圆角、阴影、过渡效果
- **清晰排版**：优化字体、行高、间距
- **响应式布局**：适配桌面和移动设备
- **深色模式**：支持系统深色模式
- **交互反馈**：悬停效果、加载动画

## 注意事项

1. **性能优化**：
   - 避免频繁 DOM 操作
   - 使用 MutationObserver 而非定时轮询
   - 延迟执行非关键操作

2. **兼容性**：
   - 测试不同浏览器
   - 处理 arXiv 网站结构变化
   - 保持原功能完整性

3. **用户体验**：
   - 提供流畅的视觉效果
   - 确保内容加载速度
   - 保持与原网站的功能一致性

4. **法律合规**：
   - 不修改原始数据
   - 不缓存敏感信息
   - 遵守 arXiv 使用条款

## 常见问题

### 扩展不生效
- 检查扩展是否已启用
- 确认 arXiv 网站 URL 是否匹配
- 尝试重新加载扩展

### 样式冲突
- 检查 CSS 选择器优先级
- 确保样式不会覆盖关键功能
- 测试不同页面的显示效果

### 性能问题
- 检查是否有过多的 DOM 操作
- 优化 MutationObserver 配置
- 减少不必要的计算
